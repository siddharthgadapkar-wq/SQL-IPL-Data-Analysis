# 🏏 IPL Data Analysis using SQL

## 📊 Project Overview
This project analyzes IPL player data using SQL queries to extract meaningful insights about player performance, team distribution, and salary trends.

## 🔍 Key Concepts Used
- Subqueries
- Aggregate Functions (MAX, AVG, COUNT)
- GROUP BY & HAVING
- Filtering using IN, NOT IN, ANY

## 📌 Key Queries Implemented
- Highest Run Scorer
- Players Above Average Runs
- Players from Same Team
- Teams with More Than 3 Players (using COUNT)
- Players Not Bowlers (Subquery)
- Players with Runs Greater Than Bowlers

## 📊 Key Insights
- Top-performing players are not always the highest paid
- Some players outperform bowlers significantly in runs
- Teams with higher player counts show strong distribution
- SQL helps in solving complex real-world filtering problems
